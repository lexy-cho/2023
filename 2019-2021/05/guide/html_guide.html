E-best Publish Guide

Document 구조

본 파일은 e-best 프로젝트 퍼블리싱 영역의 가이드 입니다.

1. Document Type

<!DOCTYPE html>
<html lang="ko">

2. HTML Head

모바일대응
<meta name="viewport" content="width=device-width,initial-scale=1.0,minimum-scale=1.0,maximum-scale=1.0,user-scalable=no" />
<meta name="format-detection" content="telephone=no"/>

모든 파일은 head 에서 공통 혹은 부분공통적으로 사용되는 css및 js 파일을 불러온다.

<link rel="stylesheet" type="text/css" href="../css/파일명.css" />
<script type="text/javascript" src="../js/파일명.js"></script>

3. Document Structure

DIV 레이아웃 사용
- 공통 : #wrap, #header(#headTop, #headSub), #container .content
- sub 공통 : .tbContainer(상단노랑색라인) .tbContainer(상단진한회색) .searchField .customerDtlBox
  1. 공통 레이아웃
    • #wrap : 전체 레이아웃
    • #header
      • #headTop: 네비게이션, 메인로고, 홈페이지 바로이동
      • #headSub: 현재화면이름, 접속된유저이름, 현재세션상황, 로그아웃
    • #container .contents : 컨텐츠
  2. sub 폼
    • sub 폼
      • .tbContainer 리스트(상단노랑색라인)
      • .tbContainer 리스트(상단진한회색)
      • .searchField (검색박스)
      • .customerDtlBox (여닫기적용데이터)
  3. 견적관리
    • .estimateWrap (내용영역)
    • .estimateBtn (하단 버튼영역)
    • .estSummary (네비게이션)
  4. 발송관리
    • .sendDetail (내용 영역)
    • .sendBtn (하단 버튼영역)
  5. 통합게시판
    • .boardDetail (내용영역)
  6. 통계
    • .commissionLayout (내용영역)
