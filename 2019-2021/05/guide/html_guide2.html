CSS 구조

1. CSS Structure

  • bbs.css : 견적/계출/계약을 제외한 모든 화면
  • common.css : 공통 레이아웃 / 컴포넌트 / 팝업 정의 / 공통 반응형 대응
  • contract.css : 전자계약 및 서류등록 (1,2차)
  • customer.css : 고객관리, 배분이관관리, MCA관리
  • duDatePicker.css : 데이터픽커
  • inview.css : 네이티브 인앱 브라우저 일 경우
  • jquery.mCustomScrollbar.min.css : mCustomScrollbar
  • preview.css : preview
  • process.css : 견적, 계출현황, 게시판, 재고조회, 통계
  • responsive.css : 모바일 대응 서브 별 상세 레이아웃
  • style.css : style custom version
  • swiper.min.css : 스와이프 css
  • tree.css : 폴더레이아웃
